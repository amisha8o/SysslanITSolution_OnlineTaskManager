# 📌 Online Task Manager Project

## 🚀 Project Overview
This is a simple Online Task Manager Web Application developed to help users manage their daily tasks efficiently. Users can add, view, update, and delete tasks in a clean and user-friendly interface.

This project is mainly built for learning frontend development and understanding how task management systems work.

---

## ✨ Features
- ➕ Add new tasks easily  
- 📋 View all tasks in a list format  
- 📝 Update existing tasks  
- ❌ Delete completed or unwanted tasks  
- 💾 Simple and fast UI interaction  
- 📱 Clean and responsive design  

---

## 🛠️ Tech Stack
- HTML5 – Structure  
- CSS3 – Styling  
- JavaScript – Functionality  

(No backend / database used in this version)
